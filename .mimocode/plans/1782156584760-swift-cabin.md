## Issue 50: Fix Lint Warning

**File:** `CourseGuide/frontend/src/App.tsx` (lines 246-260)

Move `frames` array to module scope (before `Spinner` function) so it's not recreated every render:

```tsx
// Add before function Spinner (around line 246)
const spinnerFrames = ['/', '—', '\\', '|']

function Spinner() {
  const [frame, setFrame] = useState(0)
  useEffect(() => {
    const interval = setInterval(() => {
      setFrame(prev => (prev + 1) % spinnerFrames.length)
    }, 150)
    return () => clearInterval(interval)
  }, [])
  return <span className="inline-block w-4">{spinnerFrames[frame]}</span>
}
```
